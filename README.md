# Open Science Git Activity

Welcome. In this activity, you will practise a simple Git workflow using GitHub Codespaces.

Questions:
1. What is one barrier to open science in psychology?
2. How could Git support transparent research?
3. What is one concern you have about using Git?

What we will do?:
- Pull (pick apples each from a different branch).
- Add (clean and sort your apples).
- Commit (seal up the final set of apples).
- Push (put the apples together and juice them).
